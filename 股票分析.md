# 完整的分析检查清单
当你分析任何一家A股公司时，可以用这个清单来系统性地评估：

## 第一步：财务健康度
1. 扣非净利润是否为正？（排除一次性收益干扰）

2. 经营现金流是否为正？（利润是否有现金支撑）

3. 资产负债率是否过高？（>70%需警惕）

4. 应收账款/存货是否异常增长？

## 第二步：业务竞争力
1. 收入结构是否健康？（有没有单一依赖）

2. 毛利率/净利率趋势如何？

3. 公司有没有护城河？（技术、品牌、规模、客户粘性）

4. 成长驱动力是什么？（短期、中期、长期）

## 第三步：行业与位置
1. 行业空间有多大？（天花板）

2. 公司在行业中的排名/份额？

3. 政策是支持还是打压？

## 第四步：治理与估值
1. 实控人是否可靠？质押率多高？

2. PE/PB/PS处于历史什么位置？

3. 与同行业公司相比估值贵不贵？

## 第五步：风险与机会
1. 最大的风险是什么？发生的概率？

2. 最大的催化剂是什么？何时可能发生？

3. 融资融券余额是否过高？（短期波动放大器）
