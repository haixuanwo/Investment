# 国债逆回购
本金*年利率=利息
1. 需在周末、节假日前第二天买入，否则在非工作日不会记利息，且钱不会到账
2. 需关注计息天数和到账日

| 年利率     | 1万元一天的利息                 |
| :---      | ---: |
| 13%       | 10000 * 0.013 / 365 = 0.356164 |
| 14%       | 10000 * 0.014 / 365 = 0.383561 |
| 15%       | 10000 * 0.015 / 365 = 0.410958 |
| 16%       | 10000 * 0.016 / 365 = 0.438356 |

手续费十万分之一

股票空仓时可买入短期的国债逆回购
1. 挣一点利息
2. 给自己一个冷静期，仔细的看准机会后再买入股票
